# Core Profile

SoBold's Core Profile comes with an initial set of settings, extensions and snippets to help getting started on VS Code quickly.

## Prerequisites

### Fira Code

[Fira Code](https://github.com/tonsky/FiraCode) is a free monospaced font with programming ligatures.
As it is the chosen font for Core Profile, it'll have to be installed prior to activating the profile.

Visit [GitHub Fira Code page](https://github.com/tonsky/FiraCode?tab=readme-ov-file#download--install) and install the font folder, follow
the instructions for your operating system.

### Snippets

#### PHP

- `pi`: Pretty print a variable
- `ppi`: Same as `pi` but with `display: none`
- `po`: Pretty print a string
- `ppo`: Same as `pi` but with `display: none`
- `erlo`: Error log variable
- `die`: WordPress `wp_die` snippet with a `pi` inside
- `dio`: Same as `die` but with a string.
- `dbtr`: Helper built around `debug_backtrace`

#### JavaScript

- `ready`: jQuery ready snippet
- `readyes`: jQuery ready snippet using an arrow function

#### SCSS

- `a`: BEM block with cursor on selector name terminating after the block
- `aa`: Same as `a` but with cursor terminating inside the block

<Callout type="info">
`a` is useful when adding multiple BEM blocks in succession
</Callout>
