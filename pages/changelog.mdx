# Changelog

> Inspiration for this taken from [keep a changelog](https://keepachangelog.com/en/1.0.0/)

All notable changes to this project will be documented on this page.

When submitting changes to this project, your PR must include an addition to the changelog.

The following fields should be used as sub-headings:
| Title     | Explanation               |
| --------- | ------------------------- |
| Added     | For new features          |
| Changed   | For changes to anything   |
| Removed   | For removed features      |
